🚀 HOW TO USE
═══════════════════════════════════════════════════════════════════════════════

1. Start SENTRY:
   cd C:\Users\j0w16ja\SENTRY_v2-main
   npm run dev

2. Navigate to "Project Portfolio" in the sidebar

3. View Modes:
   - 📊 Grid View (default): 3D glass cards in grid
   - 🌐 3D Orbital: Floating spheres in orbital pattern

4. Interact:
   - Click stat cards to filter by health
   - Type in search box to find projects
   - Click any card/orb for detailed view
   - In 3D mode: drag to rotate, scroll to zoom

5. Update Data:
   - Edit: C:\Users\j0w16ja\SENTRY_v2-main\data\projects.csv
   - Refresh SENTRY to see changes

🔄 UPDATING PROJECT DATA
═══════════════════════════════════════════════════════════════════════════════

To update project information:

1. Open CSV:
   C:\Users\j0w16ja\SENTRY_v2-main\data\projects.csv

2. Edit fields:
   - health: green | yellow | red
   - progress_pct: 0-100
   - next_milestone: text
   - blockers_count: number
   - Any other field

3. Save as CSV (UTF-8)

4. Refresh SENTRY in browser

IMPORTANT:
• health must be lowercase: green, yellow, or red
• progress_pct must be a number (0-100)
• Don't change project_id (unique identifier)
