# Local Executive Intel Data Workspace

This folder is for local Executive Signal Scout working data.

Most contents are ignored by git because person-level OSINT can still be sensitive even when sourced publicly.

## Folders

| Folder | Purpose |
|---|---|
| `profiles/` | Executive profile JSON files |
| `sources/` | Source metadata and extraction records |
| `signals/` | Normalized signal records |
| `runs/` | Collection run logs |
| `briefs/` | Draft local reports |
| `templates/` | Trackable JSON templates |

## Rule

Do not commit collected executive data unless a human explicitly reviews and approves the exact files.
